SEO och nyckelord

Det är viktigt att man använder sig utav nyckelord i sina h1 taggar eftersom det är dem google går på när dem skickar ut sina spindlar, men även att du nämner dina nyckelord i b.la. text som du har på sidan.
När du väljer dina nyckelord på Google för att ranka högt är det viktigt att du väljer ut specifika nyckelord som inte är lika svåra att matcha högt på.
T.ex. om du använder dig utav nyckelorder "bil" är det låg chans att du matchar högt på det,
men om du istället skulle använda dig utav nyckelorden "köp bil växjö idag" så skulle det vara mycket högre chans att bli rankad högt på de nyckelorden.

Responsiv webbdesign

Responsiv webbdesign innebär att elementen och sidan i sin helhet ändrar på sig när någonting händer, det vanligaste är vid förminskning och förstoring men det finns även andra alternativ och saker som kan hända.
Responsiv webbdesign är viktigt därför att det ger sidan en mycket bättre visuell standard med hjälp av t.ex. flexbox och css grid.
Två sätt eller metoder du kan använda dig av för att göra din sida responsiv är bl.a. Flexbox, CSS-Grid men du kan även nyttja dig utav widgth: x%;
Brytpunkter finns till för @mediascreen så att du kan ändra dina element till olika anpassade storlekar så att sidan ser lika bra ut på din datorskärm som på din telefon.
Du kan t.ex. ha en brytpunkt vid 1000px och en annan vid 800px.

Bootstrap

Bootstrap är ett tilläg kan man säga där kodare får lägga upp sin kod som vem som helst kan tanka ner helt gratis för att snabbare kunna komma igång att koda utan att behöva skriva all kod från scratch.
De två sätten du kan använda Bootstrap på din sida är antingen genom att ladda ner bootstrap från deras sida, eller så kan du inkludera bootstrap från en CDN, aka Content Delivery Network.
De två sätten skiljer sig åt genom att om du skulle ladda ner bootstrap så har du tillgång till specifika saker och kan lätt tanka över och ändra saker lite hur du vill för att undvika konflikter,
medans om du länkar till ett CDN så har du inte lika mycket kontroll på din kod vilket kan skapa hinder och problem när du jobbar med din sida.
